
## [oTree](http://demo.otree.org/) installer and launcher

## To launch

### Windows
Run `otree.bat`

### Mac/Linux
Run `otree.sh`

## Homepage
http://www.otree.org/

## About

This repo contains all the code for the oTree installer. The installer
has no dependencies and is self-contained.
